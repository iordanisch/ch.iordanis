Design Process

I get involved through all the design process.

Research

Everything starts with the comprehensive understanding of the project and its requirements. The next step is to gather information about the users using various methods such as user interviews, identifying user goals and a rbust competitor analysis.

Analysis

After gathering information and data regarding potential users, it’s time to do some user testing to see how a target group interacts with the product, see what motivates the users by create user personas, make user scenarios and user maps.

Create

Using prototyping tools such as Figma, Sketch, Invision and others is a way to give all the gathered information a basic shape by creating wireframes, lo-fi mockups and interactive prototypes and define the hierarchy of the elements.

Testing

Testing can help in understanding how successful users are with their tasks and is an essential part of the process. There are various usability testing methods to choose from in order to see if there are any flaws in the product,and see if the product meet user’s expectations.
